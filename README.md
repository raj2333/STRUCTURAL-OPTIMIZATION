# Ant Colony Optimization for Multidimensional Engineering Problems

This repository contains Python codes for solving multidimensional optimization problems using Ant Colony Optimization. The project focuses on numerical optimization, mixed-integer nonlinear programming, constraint handling, and engineering design applications.

The main objective of this work is to develop an optimization framework that can handle problems with more than four design variables and can be extended for structural optimization tasks. The code includes objective function evaluation, constraint handling, feasibility-based selection, adaptive penalty methods, local search, restart strategy, and support for future integration with finite element analysis.

This repository was developed as part of a course project on Structural optimization at NTUA.
